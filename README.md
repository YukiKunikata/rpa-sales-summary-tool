# rpa-sales-summary-toolについて
本ツールは、ユーザーが指定したフォルダ内の複数売上データ（CSV）を横断集計し、品名単位で売上を算出するRPAツールです。

## ツールの目的
Power Automate for desktopとPythonでのツール作成練習として、それぞれ作成しました。

## 作業の流れについて
①指定のフォルダ内の読込ファイル（CSV)のファイル数を確認する。
②読込ファイルがない場合、
　読込ファイルが1ファイル以上ある場合、集計処理を行う
③読込ファイル（CSV）の集計結果を新規Excelファイルにて出力する。
④ツールの実行結果をメールでユーザーに通知する。

## 実行前の事前準備
・CSVファイルは、指定のフォーマットのものを使用する。
・ユーザーは、ツール実行前、または直後に「読込ファイル（CSV）の格納元フォルダパス」と「出力先フォルダパス」をツールに設定する。

## ツール実行手順
①ユーザーが、任意のフォルダに読込ファイル（CSV）をまとめて格納しておく。
②ツールを実行する。この際、読込ファイル（CSV）を格納したフォルダのパスを、ユーザーが指定する。
③集計処理が完了するまで待機する。
④ツールの実行結果がメール通知される。
⑤出力ファイル（Excel）を確認する。
